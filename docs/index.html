Funktionen

Alles, was du brauchst, ohne das Terminal öffnen zu müssen.

Rechtsklick Kontext

Integrierte Befehle direkt im VS Code Datei-Explorer. Kein Wechseln zur Kommandozeile mehr nötig.

Lock & Unlock

Sperre binäre Dateien, um Konflikte im Team zu vermeiden, und gib sie wieder frei, wenn du fertig bist.

Force Unlock

Für den Notfall: Erzwinge das Entsperren von Dateien (Force), falls ein Prozess hängen geblieben ist.

Voraussetzungen

Stelle sicher, dass git und git-lfs auf deinem System installiert sind und sich im PATH befinden.

Nutzung

  • 1 Rechtsklick auf eine Datei im VS Code Explorer.
  • 2 Wähle Git LFS: Lock um die Datei zu sperren.
  • 3 Beobachte die Benachrichtigung unten rechts für den Erfolgsstatus.
